# BlobType Tool
By Sofía Molina // [View live project](https://sofiams.com/Blobtype/) 
### 

Blobtype is an interactive canvas-based web tool that combines generative organic blob shapes with bold custom typography. It lets users type custom text and overlay or morph it against fluid, randomized vector shapes. The interface provides a suite of real-time controls allowing you to choose from various display typefaces, tweak color palettes, adjust shape parameters, and export high-resolution canvas compositions for graphic design, branding assets, or posters.
